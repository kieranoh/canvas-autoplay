# SKKU Lecture Runner

`canvas.skku.edu` 마이 페이지의 할 일 영역에서 영상으로 보이는 항목을 모아 하나씩 열고, HTML5 영상 재생이 끝나면 다음 항목으로 이동하는 Chrome 확장자입니다.

## 설치

1. Chrome에서 `chrome://extensions`를 엽니다.
2. 오른쪽 위 `Developer mode`를 켭니다.
3. `Load unpacked`를 누릅니다.
4. 이 폴더를 선택합니다: `canvas-skku-video-autoplayer`

## 자동재생 전용 Chrome 실행

Chrome 자동재생 정책 때문에 첫 재생이 막히면 일반 Chrome 대신 아래 PowerShell 스크립트로 전용 Chrome 프로필을 실행합니다.

```powershell
.\launch-chrome-autoplay.ps1
```

이 스크립트는 `--autoplay-policy=no-user-gesture-required` 플래그와 별도 Chrome 프로필을 사용합니다.

## 사용

1. `https://canvas.skku.edu`에 로그인합니다.
2. 마이 페이지에서 할 일 목록이 보이는 화면으로 이동합니다.
3. 검색창을 비우고 `검색`을 누르면 할 일 목록의 링크를 미리봅니다.
4. 필요하면 검색창에 강의명이나 주차를 넣어 할 일 목록 안에서 필터링합니다.
5. 수집된 목록을 확인한 뒤 `시작`을 누릅니다.
6. 화면 오른쪽 아래 오버레이에서 진행 상황을 확인합니다.
7. 자동 감지가 안 되는 외부 플레이어에서는 영상 종료 후 오버레이의 `다음`을 누릅니다.

## 동작 방식과 한계

- 할 일 영역의 링크 중 `video`, `lecture`, `동영상`, `영상`, `강의`, `kaltura`, `panopto` 같은 힌트가 있는 항목을 영상 후보로 수집합니다.
- `검색`은 영상 힌트가 아니라 마이 페이지의 할 일 영역 링크를 기준으로 미리보기 목록을 만듭니다.
- 검색어를 입력하면 할 일 목록 안에서 검색어가 들어간 링크만 후보로 수집합니다.
- Canvas의 `콘텐츠로 건너 뛰기` 같은 접근성용 skip link는 후보에서 제외합니다.
- 페이지 안의 `video` 또는 `audio` 요소를 찾으면 재생 종료 이벤트를 사용해 다음 항목으로 이동합니다.
- 영상이 외부 도메인의 iframe 안에 들어 있으면 Chrome 보안 정책 때문에 확장자가 재생 시간과 종료 여부를 직접 읽지 못할 수 있습니다. 이 경우 오버레이의 `다음` 버튼으로 이어갈 수 있습니다.
- 브라우저 자동재생 정책 때문에 첫 영상에서 재생 버튼을 한 번 직접 눌러야 할 수 있습니다.
